Wetaa App

Wetaa is a multi-vendor SaaS solution designed for restaurants, bars, and cafes to streamline high-volume, in-house ordering using real-time group chat and order management technology. The app empowers vendors to manage dynamic order queues, communicate with cashiers instantly, and handle customer traffic with precision and clarity—all from a single dashboard.

Project Objective

The vision for Wetaa was to eliminate bottlenecks in crowded dine-in environments, replacing chaotic verbal orders with a smooth digital ordering and cashier communication system. The platform needed to support multiple restaurants and bars under one SaaS infrastructure, ensuring each vendor retained a unique environment, branding, and user flow while leveraging a shared scalable backend.

Product Management Challenges

  • Multi-Vendor Architecture: Building a system that could securely handle multiple vendor instances while maintaining performance integrity and user isolation.
  • Real-Time Communication: Integrating Socket.IO to facilitate frictionless, real-time chat between vendors and their cashiers, especially during high-order surges.
  • QR Code Ordering: Implementing QR-based access for customers, allowing seamless cart retrieval and order placement without requiring sign-ins or downloads.
  • Order Ownership: Ensuring clarity on order responsibility by automatically assigning attending cashiers, reducing duplicate processing and human errors.
  • Push Notification Logic: Balancing urgency and user experience by designing tiered push notification behaviors—persistent alerts for new orders vs. single ring notifications for general messages.
  • Subscription Management: Building a subscription model with trial, monthly, and yearly options, coupled with an automated deactivation system upon expiry.
